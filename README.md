# Resonance

Resonance is a NeoForge mod for Minecraft 26.1.2 that expands amethyst into a complete progression and adventure system.

## Highlights

- Resonant tools, weapons, armor, arrows, mount armor, and a protective totem.
- Resonance combat effects, vibration scars, crystal growth, specialized tool abilities, and harmonized farmland.
- Crystal forests with calcite terrain, cherry-shaped crystal trees, flora, rabbits, and armadillos.
- Shattered Echoes, Resonant Stalkers, Crystal Wraiths, Crystal Sentinels, and the multi-phase Harmonic boss.
- Harmonic Arenas, amethyst spires, safe geode placement, rare exploration loot, advancements, recipes, and recipe-book support.

## Requirements

- Minecraft 26.1.2
- NeoForge 26.1.2 or newer within the 26.1 line

## Development Build

The current private release candidate is available at [`releases/resonance-1.0.0-rc1.jar`](releases/resonance-1.0.0-rc1.jar). Its SHA-256 checksum is recorded in [`releases/SHA256SUMS.txt`](releases/SHA256SUMS.txt).

## Building

```text
./gradlew build
```

The distributable JAR is created in `build/libs`.

## Configuration

The common configuration controls Resonance damage, effect duration, Harmonic Shield cooldown, and particle visibility.
